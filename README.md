# 二年级英语单词闪卡

一个适合小学二年级学生使用的互动英语单词学习工具。

## 在线访问

[点击这里使用单词闪卡](https://你的用户名.github.io/仓库名/单词闪卡.html)

## 功能特点

- 卡片翻转动画效果
- 包含音标和中文释义
- 34个二年级上册常用单词
- 前后翻页、随机学习
- 进度跟踪
- 响应式设计，支持手机/平板/电脑
- 支持键盘快捷键操作

## 使用方法

1. 点击卡片或"翻转"按钮查看中文释义
2. 使用"上一个"/"下一个"按钮切换单词
3. 点击"随机"按钮随机抽取单词练习
4. 点击"查看所有单词"浏览完整单词表

## 键盘快捷键

- `←` 上一个单词
- `→` 下一个单词
- `空格键` 或 `回车键` 翻转卡片

## 技术栈

纯HTML + CSS + JavaScript，无需任何依赖，可离线使用。
