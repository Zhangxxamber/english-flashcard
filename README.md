# 小学学习助手 - 综合学习平台

## 📚 项目简介

一个综合性的小学学习辅助平台，涵盖语文、英语、数学三大学科，采用自主默写、互动练习等多种学习模式。

## 🎯 项目特色

- ✅ **纯前端实现**：无需后端服务器，即开即用
- ✅ **离线可用**：支持本地离线使用
- ✅ **自主学习**：自我评估模式，培养学习主动性
- ✅ **错题管理**：自动记录错题，支持针对性复习
- ✅ **进度追踪**：实时显示学习进度和正确率
- ✅ **响应式设计**：支持电脑、平板、手机多设备

## 🎓 学科模块

### 1. 语文（grade2_chinese）
- **教材**：人教版小学语文二年级上下册
- **内容类型**：写字表、识字表、词语表
- **学习模式**：自主默写（听拼音写汉字）
- **词汇量**：约 600+ 生字，200+ 词语

### 2. 英语

#### 2.1 小学英语二年级（grade2_english）
- **教材**：沪教版小学英语二年级上下册
- **学习模式**：自主默写（听发音看中文写英文）
- **词汇量**：246 个核心词汇
- **特色**：音标显示、错题本

#### 2.2 Big English 系列（big_english）
- **教材**：Big English 2nd Edition (Book 1-6)
- **学习模式**：自主默写
- **词汇量**：738 个核心词汇
- **特色**：多教材选择、分单元学习

### 3. 数学（grade2_math）
- **教材**：沪教版小学数学二年级上下册
- **学习模式**：互动练习、快速测验
- **内容**：计算练习、应用题等
- **特色**：实时反馈、错题分析

## 🚀 快速开始

1. **直接打开**：双击 `index.html` 在浏览器中打开
2. **选择学科**：在主页选择想要学习的学科
3. **开始学习**：按照提示进行学习和练习
4. **查看错题**：随时查看错题本进行复习

## 💾 数据存储说明

所有学习数据（错题记录、学习进度等）都存储在浏览器的 localStorage 中：

- **优点**：无需登录，数据本地存储，保护隐私
- **注意**：清除浏览器缓存会清空学习记录
- **备份**：建议定期导出重要数据（如有需要）

## 🎨 设计特色

- **渐变背景**：使用柔和的紫色渐变，保护视力
- **大字体**：适合小学生阅读习惯
- **卡片布局**：信息层次清晰，易于理解
- **动画效果**：流畅的过渡动画，提升使用体验
- **响应式**：自适应不同屏幕尺寸

## 🔧 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **音频**：Web Speech API（语音合成）
- **存储**：localStorage（本地存储）
- **动画**：CSS3 Animations + Transitions
- **图标**：Emoji 表情符号

## 🎯 学习建议

1. **循序渐进**：从简单到复杂，逐步提升难度
2. **坚持练习**：每天固定时间练习，形成习惯
3. **重视错题**：及时复习错题本，巩固薄弱环节
4. **自我评估**：诚实评估掌握情况，不要自欺欺人
5. **劳逸结合**：注意休息，保护视力
