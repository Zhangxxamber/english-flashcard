# 📚 Big English 1 单词闪卡

适合小学生使用的 Big English 1 教材配套单词学习工具，包含213个核心词汇，支持基础版和发音版两种学习模式。

## ✨ 功能特点

### 📖 基础版
- ✅ 213个核心词汇，覆盖入门到第9单元
- ✅ 精美卡片翻转效果
- ✅ 单元筛选功能
- ✅ 随机学习模式
- ✅ 键盘快捷键支持
- ✅ 学习进度显示
- ✅ 完整音标和例句

### 🔊 发音版（进阶版）
- ✅ 包含基础版所有功能
- ✅ 浏览器内置真人语音朗读
- ✅ 自动发音切换
- ✅ 三档语速调节（慢速/正常/快速）
- ✅ 大型发音按钮
- ✅ 键盘S键快速发音

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `←` | 上一个单词 |
| `→` | 下一个单词 |
| `空格` / `回车` | 翻转卡片 |
| `S` | 发音（仅发音版） |

## 🎯 使用建议

1. **初学者**：从基础版开始，专注于词汇记忆
2. **进阶学习**：使用发音版，同步提升听力和发音
3. **按单元学习**：使用单元筛选功能，循序渐进
4. **随机复习**：定期使用随机模式巩固记忆

## 📥 本地使用

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html`
3. 选择适合的版本开始学习
