## 🎯 学习路径设计

```
第1阶段：基础版 (认识)
   ↓
看单词 → 记意思 → 看例句 → 重复学习
   ↓
掌握单词的形态、意思和用法

第2阶段：发音版 (听说)
   ↓
听发音 → 跟读 → 调整语速 → 反复练习
   ↓
掌握单词的正确发音和听力识别

第3阶段：默写版 (写作)
   ↓
看中文 → 回忆拼写 → 输入验证 → 错题重练
   ↓
掌握单词的拼写和主动回忆

```
