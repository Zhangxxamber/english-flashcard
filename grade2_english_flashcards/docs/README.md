# 沪教版小学英语二年级闪卡系统

## 📚 项目简介

这是一个专为小学二年级学生设计的英语学习工具，基于**沪教版小学英语二年级上下册**教材开发。通过趣味互动的方式，帮助孩子们轻松掌握英语单词、短语和句子。

## ✨ 特色功能

### 🎧 听选版（推荐首选）
- **听力训练**：听英文，选中文意思
- **4选1选择题**：降低难度，适合初学者
- **真人发音**：标准美式英语朗读
- **即时反馈**：答对答错立即知道
- **正确率统计**：实时追踪学习效果
- **单元筛选**：可按单元练习

### 🃏 闪卡版
- **卡片翻转**：点击翻转查看中英文
- **真人发音**：支持语音朗读
- **语速调节**：慢速/正常/快速三档
- **随机模式**：打乱顺序记忆更牢固
- **键盘控制**：支持方向键和快捷键

### ✍️ 默写版（挑战模式）
- **看中文写英文**：真正考验拼写能力
- **智能提示**：首字母提示+语音提示
- **实时验证**：输入完成即时检查
- **详细反馈**：显示正确答案和错误分析
- **进度追踪**：记录正确率

## ⌨️ 快捷键

### 闪卡版
- `←` / `→` : 上一个/下一个
- `空格` / `回车` : 翻转卡片
- `S` : 发音

### 默写版
- `回车` : 检查答案/下一题

## 💡 使用技巧

1. **每天10分钟**：持续学习效果更好
2. **先听后说**：从听选版开始，建立语感
3. **循环复习**：定期回顾已学内容
4. **游戏化学习**：把练习当游戏，减轻压力
5. **家长陪伴**：低年级学生需要家长引导
