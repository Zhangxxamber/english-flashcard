# 小学英语二年级 - 自主默写系统

## 📚 项目简介

沪教版小学英语二年级（上下册）自主默写学习系统，支持听发音、看中文、自我评估的学习模式。

## 📂 目录结构

```
grade2_english/
├── index.html                    # 主页 - 学期选择界面
├── dictation.html                # 自主默写练习页面
├── data/                         # 词汇数据目录
│   ├── vocabulary_book1.js       # 上册词汇数据
│   └── vocabulary_book2.js       # 下册词汇数据
├── book/                         # 教材资源
│   ├── 沪教版小学英语二年级上册.pdf
│   ├── 沪教版小学英语二年级下册.pdf
│   ├── key_words_1.txt           # 上册关键词汇
│   └── key_words_2.txt           # 下册关键词汇
├── docs/                         # 项目文档
├── 自主听写模式说明.md           # 功能说明文档
├── 错题本功能说明.md             # 错题本功能文档
└── README.md                     # 本文档
```

## 🎯 功能特点

- **自主默写模式**：听发音 → 看中文 → 显示答案 → 自我评估
- **双学期支持**：上册（Starter + 10单元）、下册（10单元）
- **错题本功能**：自动记录错题，支持重练
- **音标显示**：支持国际音标查看
- **提前返回保存**：支持练习中途返回，自动保存错题
- **进度追踪**：实时显示学习进度和正确率

## 🚀 使用方法

1. 打开 `index.html` 选择学期（上册/下册）
2. 点击"开始练习"进入自主默写
3. 点击播放按钮听单词发音
4. 查看音标和中文意思
5. 在纸上写出英文单词
6. 点击"💡 显示答案"查看正确答案
7. 根据掌握情况选择：
   - ✅ **会了**：表示已掌握
   - ❌ **还不会**：自动加入错题本
8. 完成后可在首页点击"查看错题本"复习

## 💾 数据存储

- 错题数据存储在浏览器 localStorage 中
- 键名格式：`wrongItems_book1`（上册）、`wrongItems_book2`（下册）
- 清除浏览器缓存会清空错题记录
