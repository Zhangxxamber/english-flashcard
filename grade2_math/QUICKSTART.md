# 🚀 二年级数学计算练习 - 快速开始指南

## 📖 5分钟快速上手

### 第一步：打开系统

有两种方式进入系统：

**方式1：从主平台进入（推荐）**
```
1. 打开 english-flashcard/index.html
2. 找到"沪教版小学数学二年级"卡片
3. 点击"进入练习 →"按钮
```

**方式2：直接访问**
```
直接打开 grade2_math_flashcards/index.html
```

**快捷键**
```
在主平台页面按数字键 3
```

---

### 第二步：选择学期

进入后，你会看到学期选择器：

```
📚 选择学期
[ 上学期 ]  [ 下学期 ]
```

- **上学期**：加减乘法、人民币计算（6种类型）
- **下学期**：除法、三位数运算、时间计算（8种类型）

点击对应按钮即可切换。

---

### 第三步：选择练习方式

系统提供两种练习模式：

#### 模式1：📝 练习模式（推荐新手）

**特点**：逐题练习，即时反馈，有解题步骤

**使用方法**：
1. 点击"练习模式"卡片
2. 页面滚动到计算类型选择区
3. 选择一个计算类型（如"100以内加法"）
4. 开始练习！

**练习流程**：
```
看题目 → 输入答案 → 提交
    ↓
答对：✅ 绿色提示 + 音效 + 鼓励语音
答错：❌ 红色提示 + 音效 + 解题步骤
    ↓
点击"下一题"继续
    ↓
完成20题后显示成绩
```

#### 模式2：⚡ 速算测验（适合巩固）

**特点**：批量答题，计时挑战，错题回顾

**使用方法**：
1. 点击"速算测验"卡片
2. 选择计算类型
3. 选择题目数量（10/20/30/50题）
4. 点击"开始测验"
5. 计时答题
6. 提交后查看成绩和错题

---

### 第四步：开始练习

#### 如果选择了"练习模式"：

```
1. 看到题目：52 + 28 = ?
2. 在输入框中输入答案：80
3. 点击"提交答案"（或按回车键）
4. 查看反馈：
   - ✅ 答对：绿色提示 + 欢快音效
   - ❌ 答错：红色提示 + 显示正确答案和步骤
5. 点击"下一题"
6. 重复上述步骤...
```

#### 如果选择了"速算测验"：

```
1. 看到题目列表（如20题）
2. 从第1题开始，逐个输入答案
3. 按回车键自动跳到下一题
4. 完成所有题目后点击"提交测验"
5. 查看成绩：
   - 用时
   - 正确数
   - 正确率
   - 错题详情
```

## 💡 使用技巧

### 1. 连击系统

连续答对会触发特效：

- 🔥 **3连击** - 屏幕中央显示"3连击！"
- ⚡ **5连击** - 显示"5连击！"
- 🌟 **10连击** - 显示"10连击！"
- 🏆 **15连击** - 显示"超神！15连击！"

**目标**：挑战更高连击记录！

### 2. 音效开关

如果觉得音效太吵，可以关闭：

```
点击右上角的 🔊 按钮
变成 🔇 就是关闭了
```

设置会自动保存，下次打开仍然有效。

### 3. 查看解题步骤

答错题目后，会显示解题步骤：

```
💡 解题步骤：
• 个位：2不够减8，从十位借1
• 个位：12 - 8 = 4
• 十位：4 - 2 = 2
• 答案：24
```

认真看步骤，理解算法！

### 4. 快捷键

- **回车键**：提交答案 / 跳到下一题
- **数字键 3**（主平台）：快速进入数学练习

## ❓ 常见问题

### Q1: 为什么没有声音？

**A**: 检查以下几点：
1. 音效开关是否打开（右上角🔊）
2. 电脑/手机音量是否打开
3. 浏览器是否允许播放声音

### Q2: 如何重做错题？

**A**: 
- 练习模式：记住错题类型，重新选择该类型练习
- 速算测验：完成后查看错题回顾，记录下来
- （错题本功能开发中）

### Q3: 题目会重复吗？

**A**: 题目是随机生成的，理论上会有重复，但概率很低。系统可以生成成千上万道不同的题目。

### Q4: 可以选择题目难度吗？

**A**: 目前每种计算类型有固定的难度设定（简单/中等/困难）。未来版本会增加自定义难度功能。

### Q5: 手机上能用吗？

**A**: 可以！系统支持手机浏览器，界面会自动适配。

## 🎓 学习建议

### 每日练习计划

**初学者（第1-2周）：**
```
周一～周五：
- 上学期 → 100以内加法 → 20题
- 用时：10-15分钟

周末：
- 速算测验 → 加法 → 30题
- 目标：正确率 ≥ 90%
```

**进阶者（第3-4周）：**
```
周一～周五：
- 上学期 → 100以内减法 → 20题
- 或 表内乘法 → 20题
- 用时：15-20分钟

周末：
- 速算测验 → 混合运算 → 50题
- 目标：正确率 ≥ 85%
```

**挑战者（第5周+）：**
```
周一～周五：
- 下学期 → 有余数除法 → 30题
- 或 四则混合运算 → 30题
- 用时：20-30分钟

周末：
- 速算测验 → 全部类型轮流
- 目标：正确率 ≥ 95%，连击 ≥ 10
```
