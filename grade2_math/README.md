# 二年级数学计算练习系统

沪教版 · 上下学期完整覆盖 · 无限题库自动生成

---

## ✨ 功能特色

### 🎯 核心功能

1. **练习模式** - 逐题练习，即时反馈
   - 自动生成题目，无限练习
   - 实时批改，即时反馈
   - 错题显示解题步骤
   - 连击奖励系统

2. **速算测验** - 限时挑战，检验速度
   - 自选题目数量（10/20/30/50题）
   - 计时功能
   - 批量答题
   - 错题回顾

3. **音效系统** - 趣味互动
   - 答对/答错音效反馈
   - 鼓励语音播报
   - 连击特效提示
   - 可开关设置

## 🎮 使用方法

### 1. 进入系统

```
从主平台进入 → 选择"沪教版小学数学二年级"
或直接访问：grade2_math/index.html
```

### 2. 选择学期

- 点击"上学期"或"下学期"按钮
- 查看对应学期的计算类型

### 3. 开始练习

**方式一：练习模式**
1. 点击"练习模式"卡片
2. 选择想要练习的计算类型
3. 开始逐题练习
4. 输入答案后点击"提交"
5. 查看反馈和解题步骤
6. 点击"下一题"继续

**方式二：速算测验**
1. 点击"速算测验"卡片
2. 选择计算类型和题目数量
3. 点击"开始测验"
4. 计时答题
5. 提交后查看成绩和错题

## 💡 学习建议

### 日常练习

1. **每天练习 15-20 分钟**
   - 选择 1-2 种计算类型
   - 完成 20-30 题

2. **循序渐进**
   - 先练简单类型（加法、减法）
   - 再练中等类型（乘法、除法）
   - 最后挑战混合运算

3. **错题重练**
   - 注意查看解题步骤
   - 理解运算规则
   - 重复练习同类题型

### 速算测验

1. **每周测验 2-3 次**
   - 检验练习成果
   - 提升计算速度

2. **挑战自己**
   - 逐步增加题目数量
   - 尝试缩短答题时间
   - 追求更高正确率
