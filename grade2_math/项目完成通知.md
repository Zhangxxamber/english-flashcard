## 💻 技术架构

### 前端技术

```
HTML5 + CSS3 + JavaScript (ES6+)
├── 面向对象编程（类继承）
├── Web Audio API（音效系统）
├── SpeechSynthesis API（语音播报）
├── localStorage（设置存储）
└── 响应式设计（移动端适配）
```

### 核心算法

```javascript
题目生成器
├── MathQuestionGenerator（基础类）
│   ├── 加法生成算法
│   ├── 减法生成算法（避免负数）
│   ├── 乘法生成算法
│   ├── 混合运算生成算法
│   └── 人民币计算算法
│
└── MathQuestionGeneratorS2（扩展类）
    ├── 继承所有基础方法
    ├── 除法生成算法
    ├── 有余数除法算法（余数<除数）
    ├── 乘除混合算法
    ├── 数的认识算法
    └── 时间计算算法
```
