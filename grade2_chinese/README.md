# 小学语文二年级 - 自主默写系统

## 📚 项目简介

人教版小学语文二年级（上下册）自主默写学习系统，支持写字表、识字表、词语表三种内容类型。

## 🎯 功能特点

- **自主默写模式**：听拼音 → 手写汉字 → 自我评估
- **三种内容类型**：
  - 📝 写字表：需要会写的生字
  - 👀 识字表：只需要认识的生字
  - 📚 词语表：重点词语
- **两个学期**：上学期、下学期
- **错题本功能**：自动记录错题，支持重练
- **课文顺序**：按课文顺序练习，巩固课堂知识
- **提前返回保存**：支持练习中途返回，自动保存错题

## 🚀 使用方法

1. 打开 `index.html` 选择学期（上/下）和内容类型
2. 点击"开始练习"进入自主默写
3. 听拼音，在纸上写出汉字
4. 点击"显示答案"查看正确答案
5. 根据书写情况选择"会了"或"还不会"
6. 错题自动记录到错题本
7. 随时可以点击"查看错题本"复习错题

## 💾 数据存储

- 错题数据存储在浏览器 localStorage 中
- 键名格式：`wrongItems_sX_contentType`
  - s1 = 上学期，s2 = 下学期
  - contentType = xiezibiao/shizibiao/ciyubiao
- 示例：`wrongItems_s1_xiezibiao`（上学期写字表错题）
